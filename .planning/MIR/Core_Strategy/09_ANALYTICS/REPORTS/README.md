## File Naming Convention

```
Weekly:   [campaign_id]-weekly-[YYYY-WNN].md
Monthly:  [campaign_id]-monthly-[YYYY-MM].md
Post-mortem: [campaign_id]-postmortem.md
Ad hoc:   [campaign_id]-[description]-[YYYY-MM-DD].md
```

## Examples

```
acme-lead-gen-2503-weekly-2025-W12.md
acme-lead-gen-2503-monthly-2025-03.md
acme-lead-gen-2503-postmortem.md
```
