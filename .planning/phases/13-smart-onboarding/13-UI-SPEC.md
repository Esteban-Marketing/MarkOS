# Phase 13 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | HTML/CSS/JS (Vanilla) |
| Preset | Linear/Vercel-inspired |
| Component library | none (custom CSS mapped to onboarding.css) |
| Icon library | Heroicons (SVG inline) |
| Font | Inter or system-ui |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing (input to helper) |
| md | 16px | Default element spacing (form groups) |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps, card padding |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing (top/bottom) |

Exceptions: none

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0A0A0A | App Background, major surfaces (Dark Mode) |
| Secondary (30%) | #171717 | Cards, modal backgrounds, input fields |
| Accent (10%) | var(--accent) | Primary CTA buttons, Confidence Indicators (AI Spark) |
| Destructive | #EF4444 | Destructive actions, Error states, Red Confidence (<50%) |
| Warning | #EAB308 | Yellow Confidence (50-84%) |
| Success | #22C55E | Green Confidence (>=85%) |

Accent reserved for: Primary action buttons (Step 0 submit, form continue), AI Spark popover triggers, active state indicators. Never used for decorative backgrounds or static text.
