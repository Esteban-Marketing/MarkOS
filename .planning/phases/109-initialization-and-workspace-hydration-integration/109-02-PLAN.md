### Task 1: Extend `generateSkeletons` signature and wire overlaySlug

**File:** `onboarding/backend/agents/skeleton-generator.cjs`

**Action:** Modify `generateSkeletons` to accept an optional 5th parameter `packSelection = null`. Extract `overlaySlug` from it. Pass `overlaySlug` as the 4th argument to the existing `resolveSkeleton(discipline, businessModel, templatesBasePath)` call.

**Details:**

Current signature (line ~44):

New signature:
```javascript
async function generateSkeletons(seed, approvedDrafts, outputBasePath = MARKOS_LOCAL_DIR, templatesBasePath = TEMPLATES_DIR, packSelection = null) {
```

After the existing `const slug = getModelSlug(businessModel);` line, add:
```javascript

```

Update the `resolveSkeleton` call inside the `DISCIPLINES.map` callback:

Current:

New:

All other logic — `buildFrontmatter`, `interpolatePainPoints`, `fs.writeFileSync`, error handling — remains UNCHANGED.

No changes to `module.exports`.
