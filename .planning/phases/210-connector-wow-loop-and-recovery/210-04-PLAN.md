# Plan 210-04 - First Live-Data Wow Audit

**Phase:** 210 - Connector Wow Loop and Recovery  
**Status:** Planned  
**Requirements:** CONN-03, CONN-05, LOOP-01, QA-01..15

## Objective

Produce a useful audit from live data within the target activation window.

## Scope

- Implement or plan the first GA4/GSC or equivalent analytics/search audit path.
- Define fallback paths for CRM-led or social-led tenants.
- Link audit findings to briefs, tasks, evidence, and weekly narrative inputs.
- Track time-to-connect and time-to-wow.

## Acceptance Criteria

- First live connector can be set up in under 10 minutes when credentials are available.
- First useful audit appears within 30 minutes or creates a precise recovery task.
- Audit findings create tasks, briefs, or evidence, not passive charts.

## Verification

- Activation tests cover success, no-data, stale-data, and connector failure states.
