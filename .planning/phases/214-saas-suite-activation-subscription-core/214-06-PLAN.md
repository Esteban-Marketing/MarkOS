# Plan 214-06 - SaaS Core UI/API and Growth-Mode Extension Point

**Phase:** 214 - SaaS Suite Activation and Subscription Core  
**Status:** Planned  
**Requirements:** SAS-01..04, SG-01, QA-01..15

## Objective

Expose SaaS activation and subscription core without activating the full SaaS Growth OS.

## Scope

- Define activation-gated UI/API surfaces for SaaS profile, plans, subscriptions, and lifecycle tasks.
- Add placeholder fields for future SaaS growth mode without making PLG/ABM/referral/community modules active.
- Link unresolved pricing to Pricing Engine placeholder policy.

## Acceptance Criteria

- SaaS core surfaces appear only for activated SaaS tenants.
- Growth-mode fields are stored as extension points only.
- No post-217 growth module becomes runnable from this phase.

## Verification

- UI/API gating tests and growth-module non-activation tests are specified.
