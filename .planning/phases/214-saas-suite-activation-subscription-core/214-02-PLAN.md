# Plan 214-02 - SaaS Profile, Plan, Subscription, and Lifecycle Schema

**Phase:** 214 - SaaS Suite Activation and Subscription Core  
**Status:** Planned  
**Requirements:** SAS-02, SAS-03, SAS-04, QA-01..15

## Objective

Create the durable core objects for SaaS activation and subscription management.

## Scope

- Define `SaaSSuiteActivation`, `SaaSProfile`, `SaaSPlan`, `SaaSSubscription`, and `SaaSSubscriptionEvent`.
- Add RLS, audit, deletion/export, and support visibility posture.
- Reference PricingRecommendation records or placeholder state for prices/packages.

## Acceptance Criteria

- SaaS plan and subscription objects are tenant-scoped and auditable.
- Plan pricing cannot become canonical without Pricing Engine context.
- Lifecycle events are append-only.

## Verification

- Contract tests cover plan, subscription, event, and RLS shape.
