# Plan 205-06 — Pricing Engine Operator UI + Billing Placeholder Rewrite

## Objective

Ship 8 operator pages (dashboard, matrix, cost-model wizard, recommendations index + detail, price tests, watch list, alerts) + 4 reusable components, rewrite `app/(markos)/settings/billing/page-shell.tsx` and `api/billing/tenant-summary.js` to be placeholder-safe (no hard-coded `Growth Monthly` / `$150.00`), and add the pricing-decision lineage panel on admin billing. Enforce grep-residue gate: `app/(markos)/settings/billing/**` and `api/billing/**` contain zero hard-coded tier names or dollar amounts after this plan lands.

## Current code path this migrates

- `app/(markos)/settings/billing/page-shell.tsx` — currently renders `"Growth Monthly"` title + `usageRows` + `invoices` + `$150.00` total. Rewritten to prop-drive from `tenant-summary.js`.
- `app/(markos)/admin/billing/page.tsx` — operator view; gains a collapsible "Pricing decision lineage" panel fed by linked `PricingRecommendation.recommendation_id`.
- `api/billing/tenant-summary.js` — currently returns static sample. Rewritten to call `readTenantRecommendations` (Plan 205-05) + `readTenantPricingFloor`; returns placeholder when no approved recommendation.
- Storybook fixtures — reshaped to `placeholder_state | approved_state | hold_state` variants instead of a baked `Growth Monthly` variant.

## must_haves

- 8 pages live under `app/(markos)/pricing/`.
- 4 reusable components under `components/markos/pricing/`.
- `page-shell.tsx` and `tenant-summary.js` emit `{{MARKOS_PRICING_ENGINE_PENDING}}` when no approved recommendation; visibly labeled as placeholder state (uses `PlaceholderBanner`).
- Pricing decision lineage panel visible on admin billing page.
- F-117 YAML populated with UI-facing read contracts for dashboard, matrix, cards.
- Grep-residue gate: `grep -rE '(Growth Monthly|\\bStarter\\b|\\bProfessional\\b|\\$[0-9]+)' app/\\(markos\\)/settings/billing/ api/billing/ lib/markos/billing/` returns 0 matches outside allowed exclusions.
- 5 Playwright specs green.
- Storybook variants per `RecommendationState` value.
- Accessibility: `axe-playwright` run green on all 8 pages (QA-01).
- No `stripe` import in any file in this plan.
