# Plan 212-03 - LiteracyUpdateCandidate Queue and Admin Review

**Phase:** 212 - Learning and Literacy Evolution  
**Status:** Planned  
**Requirements:** LRN-04, LRN-05, QA-01..15

## Objective

Promote learning to central literacy only through reviewed candidates.

## Scope

- Define LiteracyUpdateCandidate with anonymized pattern, source overlays, sample size, evidence summary, reviewer, and decision.
- Add admin review states: proposed, needs evidence, approved, rejected, promoted.
- Define promotion targets in literacy/PageIndex.

## Acceptance Criteria

- Central literacy cannot be updated directly from one tenant's raw outcome.
- Admin review is required before promotion.
- Rejections and revisions preserve reason and evidence.

## Verification

- Admin workflow tests cover approve, reject, revise, and promote.
