# Plan 217-05 - `markos-saas` MCP Tool Family

**Phase:** 217 - SaaS Revenue Intelligence, SAS Agents, API/MCP/UI Readiness  
**Status:** Planned  
**Requirements:** SAS-10, MCP-01, RUN-07, QA-01..15

## Objective

Expose SaaS Suite workflows to agents through the existing MCP safety substrate.

## Scope

- Define logical `markos-saas` MCP tools for subscriptions, health, support, product usage, revenue, and agent summaries.
- Reuse tenant-bound sessions, scopes, budget caps, approval tokens, and cost metering.
- Mark mutating tools as approval-required.

## Acceptance Criteria

- MCP tools are tenant-scoped and activation-gated.
- Cost and approval behavior matches existing MCP pipeline.
- Tool schemas are generated and tested.

## Verification
