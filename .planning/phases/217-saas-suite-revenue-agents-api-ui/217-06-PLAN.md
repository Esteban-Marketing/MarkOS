**Status:** Planned  
**Requirements:** SAS-10, SG-01..12, TASK-02..05, QA-01..15

Expose SaaS Suite as a coherent UI surface while keeping doc 17 growth modules gated for later work.

- Define SaaS navigation for Overview, Subscriptions, Plans, Invoices, Health, Support, Product Usage, Revenue, and Agents.
