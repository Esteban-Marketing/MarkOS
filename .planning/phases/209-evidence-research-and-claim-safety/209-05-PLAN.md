# Plan 209-05 - Research Context Reuse

**Phase:** 209 - Evidence, Research, and Claim Safety  
**Status:** Planned  
**Requirements:** EVD-04, RUN-02, QA-01..15

## Objective

Make agents reuse fresh research context before spending time and money on duplicate research.

## Scope

- Define research context lookup by tenant, topic, source class, freshness, and claim type.
- Add run metadata for reused evidence versus new research.
- Define cache invalidation and refresh triggers.
- Preserve cost visibility for reused and newly gathered context.

## Acceptance Criteria

- Agents can find fresh context and know when it is insufficient.
- Reuse decisions are recorded on AgentRun.
- Stale or weak context triggers research tasks or re-research.

## Verification

- Retrieval tests cover reuse, expiry, contradiction, and missing-context paths.
