# Plan 08-02: Template Tokenization & Categorization

## Objective
Implement strict token taxonomy into all templates. Add YAML frontmatter tokens to define schema versions and roles to optimize autonomous retrieval.

## Tasks
<task id="08-02-T1">
  <title>Inject Token Taxonomy Meta-Tags</title>
  <action>
    Add taxonomy blocks (`<!-- mgsd-token: ... -->`) to `MIR`, `MSP`, and `RESEARCH` template headers. This allows an orchestration agent to `grep` directly for tokens rather than guessing the structure or searching for free text.
  </action>
</task>

<task id="08-02-T2">
  <title>Agent Role Token Mapping</title>
  <action>
    Update agent persona definitions (`mgsd-researcher`, `mgsd-auditor`) to include required references and search heuristics based on these new token formats.
  </action>
</task>
