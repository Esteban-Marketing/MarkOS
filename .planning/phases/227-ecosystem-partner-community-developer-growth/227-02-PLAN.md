| plugin_manifest_id → IntegrationListing render | Manifest drift exposes stale capability claims (Pitfall 1) |
| cron miss → expired cert exposure | If recertification cron misses a day, query-time TRIPLE gate must still hide listing (RH5 / D-75) |
