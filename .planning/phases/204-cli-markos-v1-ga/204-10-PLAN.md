<!-- Bump script shape: -->
```javascript
// scripts/distribution/bump-homebrew-formula.cjs
// Usage: node scripts/distribution/bump-homebrew-formula.cjs
// Reads:  package.json version
// Fetches: GET https://registry.npmjs.org/markos/<version> → parses dist.shasum (sha1) + dist.tarball
//          OR computes sha256 via HEAD on tarball + streaming hash — npm registry returns sha1 historically
//          but modern npm CLI verifies via integrity hash (sha512). Homebrew requires sha256.
//          → Download the tarball, pipe through crypto.createHash('sha256'), compare to npm integrity if present.
// Rewrites: Formula/markos.rb url + sha256 fields
// Exits: 0 on success + prints diff; 1 on error
```
