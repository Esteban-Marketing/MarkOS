# Plan 207-05 — Cost Estimate, Actual Cost, Billing, Pricing Engine Context

## Objective

Add first-class cost to every AgentRun per D-207-05: estimated cost BEFORE execution, actual cost AFTER execution, billing event emit on close, and a placeholder-safe pricing_engine_context JSONB field that Phase 205 populates later. Complete migration 105 + F-110 YAML. Wire `lib/markos/billing/usage-normalizer.ts` to consume run close events.

## Current code path this migrates

- `lib/markos/billing/usage-normalizer.ts` — existing `normalizePluginUsageEvent` stays. This plan adds a sibling `normalizeAgentRunCostEvent` that converts a run-close event into a `BillingUsageEvent` with `event_name: 'agent_run_cost'`.
- `lib/markos/mcp/cost-table.cjs` + `lib/markos/mcp/cost-meter.cjs` — existing MCP per-tool cost budget stays. This plan reuses `estimateToolCost` + `computeToolCost` from `cost-table.cjs` as primitives for run-level cost estimation.
- `lib/markos/billing/invoice-line-items.ts` — gets a new input type `AgentRunCostLineItem`.
- `supabase/migrations/89_markos_mcp_cost_window.sql` — stays; not replaced. Run-level cost is complementary (per-run), not replacing per-tenant window.

## must_haves

- Migration 105 complete: 7 columns added to `markos_agent_runs` (estimated_cost_usd_micro, actual_cost_usd_micro, cost_currency, pricing_engine_context, model, tokens_input, tokens_output).
- `lib/markos/agents/cost-estimator.ts` exports `estimateRunCost(input)` called during `createRun` in 207-02.
- `lib/markos/agents/cost-actualizer.ts` exports `actualizeRunCost(runId, { tokens_input, tokens_output, provider_cost_usd_micro })` called on run close.
- `normalizeAgentRunCostEvent` emits BillingUsageEvent on run close with `source_type: 'agent_run'`.
- `pricing_engine_context` safely accepts `{{MARKOS_PRICING_ENGINE_PENDING}}` placeholder AND valid Pricing Engine context JSON from Phase 205.
- 1 new endpoint `GET /api/tenant/runs/{run_id}/cost` exposes both estimated and actual cost + pricing context.
- F-110 YAML complete.

    ```ts
    export function normalizeAgentRunCostEvent(rawEvent: Record<string, unknown>): BillingUsageEvent {
      if (rawEvent.event_name !== 'agent_run_cost') throw new Error('BILLING_INPUT_INVALID:event_name');
      const tenantId = assertString(rawEvent.tenant_id, 'tenant_id');
      const runId = assertString(rawEvent.run_id, 'run_id');
      const amountUsdMicro = assertFiniteNumber(rawEvent.amount_usd_micro, 'amount_usd_micro');
      const measuredAt = assertMeasuredAt(rawEvent.measured_at);
      const dedupeKey = buildUsageDedupeKey({
        source_type: 'agent_run',
        tenant_id: tenantId,
        correlation_id: runId,
        unit_type: 'usd_micro',
        source_payload_ref: String(rawEvent.cost_event_id || ''),
        operation_name: String(rawEvent.event_kind || 'actual'),
      });
      return {
        event_id: buildUsageEventId(dedupeKey),
        source_type: 'agent_run',
        tenant_id: tenantId,
        correlation_id: runId,
        unit_type: 'usd_micro',
        quantity: amountUsdMicro,
        measured_at: measuredAt,
        ...toMonthPeriod(measuredAt),
        dedupe_key: dedupeKey,
      };
    }
    ```
