## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] IDE jsx-a11y linter rejects `aria-invalid={isError ? 'true' : undefined}` and `aria-busy={isSubmitting || undefined}`**
- **Found during:** Task 1 (PostToolUse:Write hook diagnostics)
- **Issue:** IDE flags `Invalid ARIA attribute value: aria-invalid="{expression}"` and `aria-busy="{expression}"` for the conditional-string + boolean-or-undefined pattern. The plan's prescribed `aria-invalid={isError ? 'true' : undefined}` (Pitfall 6 spec line 459) tripped the linter false-positive.
- **Fix:** Migrated to plain boolean `aria-invalid={isError}` and `aria-busy={isSubmitting}`. React serialises booleans to `'true'`/`'false'` strings on attribute output, which is WAI-ARIA spec-compliant. The `.c-input[aria-invalid="true"]` selector in `styles/components.css` line 159 still triggers correctly since React emits `aria-invalid="true"` when `isError === true` and omits/falsifies otherwise.
- **Files modified:** `app/(marketing)/signup/page.tsx` (2 lines: aria-invalid + aria-busy)
- **Commit:** Folded into Task 1 atomic commit `160afa3`.

**2. [Rule 3 - Blocking precedent] `npm run build` script does not exist in this repo**
- **Found during:** Pre-task review (Plan 213.2-01 SUMMARY noted same)
- **Issue:** Plan's verify block calls `npm run build`; `package.json` has no such script.
- **Fix:** Substituted `npm run build-storybook` (full Vite + tsc compile pass over all TSX surface incl. `page.tsx`, `page.stories.tsx`, all module.css imports) per Plan 213.1-03 + Plan 213.2-01 precedent.
- **Files modified:** None (verification harness only)
- **Commit:** Documented in this SUMMARY.

**3. [Rule 3 - Blocking precedent] `node --test test/auth/` (directory form) crashes on Win/Node 25**
- **Found during:** Pre-task review (Plan 213.2-01 SUMMARY noted same)
- **Issue:** Node 25's CJS resolver does not interpret bare directory path as glob root; verify block uses `node --test test/auth/`.
- **Fix:** Substituted `node --test test/auth/*.test.js` (explicit glob) — 61/61 pass.
- **Files modified:** None (verification harness only)
- **Commit:** Documented in this SUMMARY.
