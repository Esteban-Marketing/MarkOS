## Deviations from Plan

**1. [Rule 3 - Blocking] `npm run build` script does not exist in this repo**
- **Found during:** Task 1 verification (post-implementation)
- **Issue:** `package.json` has no `"build"` script; the plan's verify block explicitly calls `npm run build`.
- **Fix:** Substituted `npm run build-storybook` as build-cleanliness verifier (Plan 213.1 Plan 01-03 deviation precedent — already documented in `213.1-03-SUMMARY.md` lines 95, 171). Storybook build performs the full Vite + tsc compile pass over all TSX (`page.tsx`, `LoginCard.tsx`, story file, all module.css imports) so build-cleanliness is fully verified through the proxy. Cited in decisions frontmatter.
- **Files modified:** None (verification harness only)
- **Commit:** Documented in this SUMMARY; no code change.

**2. [Tooling - Diagnostic] `node --test test/auth/` (directory form) crashes with `MODULE_NOT_FOUND` on Node 25 + Windows**
- **Found during:** Task 2 verification
- **Issue:** Node 25's CJS resolver does not interpret a bare directory path as a glob root; the plan's verify block uses `node --test test/auth/` which fails to enumerate.
- **Fix:** Substituted `node --test test/auth/*.test.js` (explicit glob) — 61/61 pass. Same passes shipped in commit message of Task 2.
- **Files modified:** None
- **Commit:** Documented in this SUMMARY; no code change.
