## CRITICAL: Gradient Elimination

**Finding:** `page-shell.module.css` line 61–68 contained `.brandCard { background: linear-gradient(135deg, #0f766e, #155e75); }` — the only `linear-gradient` violation remaining in the entire Wave 3 settings group (RESEARCH.md key finding #2, UI-SPEC AC X-3).

**Resolution:** Replaced with:
```css
.brandCard {
  background: var(--color-surface-raised);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
  padding: var(--space-md);
  display: flex;
  align-items: center;
  gap: var(--space-md);
}
```

**Verification:** `grep -niE "linear-gradient|radial-gradient" page-shell.module.css` → 0. X-3 closed for entire wave.
