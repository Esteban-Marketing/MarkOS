## Sub-Component Extraction Decision

**Decision: Extracted `MembersPageView` to `app/(markos)/_components/MembersPageView.tsx`.**

Rationale: `page.tsx` JSX exceeds the 40 LOC threshold (full table + form + modal + notice = ~160 LOC of presentational markup). `MembersPage` is a `'use client'` component with no props — Storybook cannot control internal `useState` without extraction. Pattern mirrors `LoginCard` (Phase 213.2-01). `page.tsx` retains all fetch/handler logic; `MembersPageView` receives all state as props.
