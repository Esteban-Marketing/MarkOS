# SMS Message Crafting

> Channel: **SMS**. Stage 6 of [[Message Crafting Pipeline]]. Every draft in this channel conforms to the rules below + brand voice from [[Brand System Canon]].

## Constraints

- ≤ 160 chars per segment; 10DLC registration in US.
- Opt-out phrase (STOP) mandatory.
- Brand prefix if legally required.
- No emoji unless brand-approved.
- Short link with UTM.

## Canonical structure

1. Brand prefix (if required)
2. Name or situation
3. Value in one clause
4. Short link
5. Opt-out

## Hook patterns

- recent action reference
- direct ask
- status update

## Proof patterns

- specific number
- recent event reference

## Anti-patterns (auto-reject)

- long links
- multi-message spam
- emoji walls
- false urgency

## Voice defaults

Blunt, concise, cool unless in onboarding flow.

## Audience × pain tailoring

Every draft composes: **this channel's canonical structure × resolved [[Audience Archetype Canon|archetype]] × tagged pain from [[Pain-Point Engine]] × brand voice from [[Brand System Canon]] × optional triggers from [[Neuro Audit Canon]]**.

## Rejection criteria

- Missing archetype or pain tag → reject (Canon commandment I + II).
- Violates constraints above → reject or revise.
- Fails voice classifier or claim check → reject.
