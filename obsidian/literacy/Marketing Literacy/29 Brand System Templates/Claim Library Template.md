# Claim Library Template

> Single source of truth for what the brand is allowed to say. Every claim has evidence, scope, and decay date. Referenced by Auditor agent before publish.

## Schema

| Claim | Evidence | Scope | Owner | Last verified | Decay |
|---|---|---|---|---|---|
| | | | | | |

## Example row

| Claim | Evidence | Scope | Owner | Last verified | Decay |
|---|---|---|---|---|---|
| Cuts CAC 30% in 90 days | case-study-acme-2025-q4, case-study-bolt-2026-q1 | B2B SaaS Tier-1 | Head of Growth | 2026-04-01 | 12 months |

## Claim-type rules

- **Performance claim** — requires named evidence row with reviewable source.
- **Comparison claim** — benchmarked + date-stamped + fair comparison.
- **Regulated category** (health, finance, food, children's) — legal review required.
- **Testimonial** — FTC disclosure if compensated.
- **Superlative** ("best", "fastest", "leading") — benchmark data in-hand or strike.

## Review cadence

Quarterly. Decayed rows archive, not delete. Evidence remains discoverable.
