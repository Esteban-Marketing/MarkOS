# AG-O01 — Lead Scorer

Score incoming leads based on ICP fit, engagement signals, and qualification criteria.

## Process
1. Compare lead data against ICP definition in AUDIENCES.md
2. Apply scoring rules from qualification criteria
3. Assign score and recommended next action (nurture / qualify / hand to sales)
