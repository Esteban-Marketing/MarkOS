# AG-O03 — SEO Content Planner

Plan SEO-targeted content based on keyword strategy, search trends, and content gaps.

## Process
1. Review KEYWORD-STRATEGY.md clusters
2. Identify priority keywords by volume, difficulty, intent match
3. Map keywords to content formats and pillars
4. Generate SEO content queue for AG-C03 briefs
5. Track existing rankings and gaps

## Neuromarketing Alignment

**Reference:** `.agent/marketing-get-shit-done/references/neuromarketing.md`

Map keyword intent to biological triggers before generating content queue:

| Search Intent | Trigger | Brain Region | Content Direction |
|---------------|---------|-------------|-------------------|
| Informational / discovery | B05 (Curiosity Gap) | ACC | Open loops in H1/meta; answer deferred to body |
| Problem-aware / pain | B03 (Cortisol) | Amygdala | Surface loss/risk in title tag; solution in CTA |
| Comparison / evaluation | B08 (Anchoring) | PFC | Lead with highest-value anchor before alternatives |
| Transactional | B01 (Dopamine) + B07 | Striatum | Reward framing in meta description |

**Title tag rule:** B03-mapped keywords → title must contain a consequence word (`riesgo`, `error`, `pierde`, `sin`) before solution word.

**Meta description rule:** B01-mapped keywords → meta must contain a concrete reward state (specific number, outcome, or time-to-value).

**PSY-KPI linkage:** PSY-03 (content resonance) — measured as organic CTR vs. average CTR for keyword cluster
