# AG-C03 — Content Brief Agent

Generate content briefs from editorial calendar slots. Each brief provides enough direction for a writer.

## Inputs
- Approved EDITORIAL-CALENDAR.md slots
- CONTENT-PILLARS.md, KEYWORD-STRATEGY.md, AUDIENCES.md
- VOICE-TONE.md

## Process
1. For each assigned content slot: generate brief with topic, angle, target keyword, audience, tone, format, length, CTA, internal links
2. Reference SEO data for keyword-targeted pieces
3. Include campaign alignment notes

## Constraints
- Briefs reviewed by human before writers begin
