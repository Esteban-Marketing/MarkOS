# Marketing Pitfalls Template

Common failure patterns to check before campaign execution.

## Budget Pitfalls

- [ ] **Underspending test budget:** Insufficient data to optimize (minimum $[threshold]/day per ad set)
- [ ] **No kill conditions defined:** Campaign runs indefinitely without performance gates
- [ ] **Budget front-loaded:** Spending before learning phase completes
- [ ] **Platform diversification too early:** Spreading thin before proving one channel

## Tracking Pitfalls

- [ ] **Missing CAPI configuration:** Relying only on browser-side pixel in post-iOS14 environment
- [ ] **UTM inconsistency:** Different naming conventions across campaigns
- [ ] **No baseline metrics:** Launching without documented pre-campaign performance
- [ ] **Attribution window mismatch:** Different windows across platforms invalidating comparisons

## Creative Pitfalls

- [ ] **Message mismatch:** Ad copy promises ≠ landing page delivers
- [ ] **Frequency blindness:** Not monitoring ad fatigue / frequency caps
- [ ] **Single creative variant:** No A/B testing from launch
- [ ] **Brand inconsistency:** Different visual treatment across platforms

## Audience Pitfalls

- [ ] **Audience too broad:** Targeting everyone = targeting no one
- [ ] **Lookalike from weak seed:** Building lookalikes from low-quality source audiences
- [ ] **Platform audience overlap:** Running same audience on multiple campaigns without exclusions
- [ ] **Retargeting without sequencing:** Showing same message to all website visitors regardless of engagement level

## Strategy Pitfalls

- [ ] **Optimization for vanity metrics:** Optimizing for clicks/impressions instead of conversions
- [ ] **Premature scaling:** Scaling budget before finding winning creative/audience combination
- [ ] **Ignoring seasonality:** Not adjusting for seasonal performance patterns
- [ ] **No competitive monitoring:** Not tracking competitor activity that impacts your positioning

## Process Pitfalls

- [ ] **Manual campaign management:** Not automating repeating tasks
- [ ] **Infrequent reporting:** Not reviewing performance frequently enough to course-correct
- [ ] **Missing post-mortem:** Not documenting learnings after campaign completion
- [ ] **Siloed execution:** Disciplines running independently without coordination
