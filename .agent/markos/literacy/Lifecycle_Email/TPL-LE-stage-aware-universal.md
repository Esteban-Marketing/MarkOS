# Lifecycle Email Universal Template

Lifecycle email should feel like a well-timed operator note, not a recycled campaign blast. The tone, proof, and CTA should adapt as the user moves from first interest to long-term retention.

## EVIDENCE BASE
- Activation and retention improve when emails remove uncertainty rather than add noise.
- Stage-aware email works best when each send answers the next natural question the user has.

## CORE TACTICS
### Awareness and Consideration
Educate with calm specificity and one clear takeaway.

### Decision
Increase certainty with real proof, objection handling, and precise next-step framing.

### Onboarding
Prioritize the first win, the setup path, and the confidence-building cues that reduce abandonment.

### Retention
Reinforce value, progress, and usage momentum in a supportive tone.
