# Shared Proof Posture Guidance

Proof should intensify with commitment. The closer the buyer is to a decision, the more the message should move from claims to specifics, from abstractions to evidence, and from promises to credible outcomes.

## EVIDENCE BASE
- Trust builds faster when proof format matches the buyer's risk level.
- Light educational proof works well in Awareness, while decision-stage assets need stronger evidence like examples, numbers, or case studies.

## CORE TACTICS
### Awareness proof
Use category evidence, clear diagnosis, and simple observations.

### Consideration proof
Show mechanism, process clarity, and why the approach is more believable than the default alternative.

### Decision proof
Use credible numbers, case-study logic, implementation readiness, or product evidence that lowers commitment fear.

### Onboarding and Retention proof
Prove the first win quickly, then reinforce ongoing value so the user feels progress instead of post-purchase doubt.
