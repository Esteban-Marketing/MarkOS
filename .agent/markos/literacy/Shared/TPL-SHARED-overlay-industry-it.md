# IT & Technology Overlay

Use this overlay when the buyer is an IT decision-maker, security team, or technical procurement lead evaluating technology products, managed services, or IT solutions. Technical skepticism is the default posture — this buyer has been oversold by vendors before. Credibility comes from specificity: how the product works architecturally, what the security posture actually is, and what ROI has looked like for similar organizations. Vague benefit language is immediately identified and triggers distrust.
