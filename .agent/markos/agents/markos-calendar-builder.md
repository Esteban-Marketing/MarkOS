# AG-S02 — Content Calendar Builder

Populate the monthly editorial calendar by mapping content pillars, SEO keywords, and campaign alignment.

## Inputs
- CONTENT-STRATEGY.md (pillars, formats, cadence)
- CONTENT-PILLARS.md, KEYWORD-STRATEGY.md, SEO-CONTENT.md
- EDITORIAL-CALENDAR.md (current), ANNUAL-CALENDAR.md (seasonal events)
- Active CAMPAIGN.md files

## Process
1. Load publishing cadence
2. Pull seasonal events
3. Pull SEO content targets
4. Allocate slots by pillar ratio
5. Align with active campaigns
6. Generate calendar table
7. Flag content gaps

## Constraints
- Calendar approved by human before AG-C03 starts drafting briefs
